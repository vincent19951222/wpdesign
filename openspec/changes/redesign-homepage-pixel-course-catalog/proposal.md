## Why

当前首页更像一个浅色像素工作台，视觉重心放在工具入口和控制面板上，这和你想要的“暗色、游戏化、课程探索型首页”不一致。现在需要把首页重构成更接近像素游戏菜单和课程目录的体验，让用户进入站点时感受到“探索课程世界”，而不是“进入后台工具”。

## What Changes

- 将首页从浅色工作台式布局重构为暗色像素课程目录首页，采用深色背景、像素大标题、现代可读正文和游戏化导航结构。
- 新增首页专用的 dark pixel 视觉基线，包括深色背景、课程卡片容器、筛选 pills、搜索框、像素风 CTA 和更克制的结构边框。
- 重构首页信息架构，引入顶部导航、世界观 hero、支持方横幅或场景带、搜索与分类筛选区，以及课程卡片网格。
- 将首页卡片从“主题选择卡”重构为“课程/模块卡”，强调像素场景封面、课程标题、描述和难度标签的课程发现体验。
- 保留现有技术栈与核心应用入口，但弱化首页中偏工作台和控制面板的表达，避免继续沿用上一版的浅底像素控制台风格。

## Capabilities

### New Capabilities
- `homepage-dark-pixel-shell`: 定义首页的暗色像素化外壳，包括顶部导航、hero、支持方横幅、双字体层级、深色 token 与主 CTA 规则。
- `homepage-course-discovery-catalog`: 定义首页的搜索、分类筛选、课程卡片网格和游戏化课程浏览体验。

### Modified Capabilities

## Impact

- 受影响代码主要在首页相关 React 组件、共享样式 token、全局字体与颜色配置，以及课程/卡片类展示组件。
- 需要调整 `LandingPage`、首页用到的共享按钮与卡片样式，以及首页所依赖的数据映射方式，但不涉及 Markdown 编辑、预览、复制或微信草稿同步流程。
- 不引入新的前端框架，继续基于当前 `Vite + React + Tailwind` 体系完成实现。
